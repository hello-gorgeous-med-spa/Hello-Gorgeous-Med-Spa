# Aurora, IL Visibility — Action Plan

**Problem:** Hello Gorgeous doesn't show up for "Botox near me," "med spa Aurora," and similar Aurora searches.

**Most "near me" visibility comes from Google Business Profile (GBP), not the website.** Do the GBP steps first.

---

## 🔴 CRITICAL: Google Business Profile (Do This Week)

### 1. Add Aurora to Service Areas

**GBP → Edit profile → Service areas**

- Ensure **Aurora, IL** is listed
- Add: Aurora, Naperville, Plainfield, Oswego, Yorkville, Montgomery, Sugar Grove
- Or test removing service area (see GBP-VISIBILITY-FIXES.md) — some storefronts rank better without it

### 2. Add Individual Services (Biggest Impact)

Google matches "Botox near me" to businesses that list **Botox** in their GBP Services section.

**GBP → Edit profile → Services → Add each:**

| Service |
|---------|
| Botox |
| Dysport |
| Jeuveau |
| Dermal fillers |
| Lip filler |
| Morpheus8 |
| Morpheus8 Burst |
| Laser hair removal |
| Semaglutide |
| Tirzepatide |
| Hormone therapy |
| IV therapy |

### 3. Update Description — Include Aurora

**GBP → Edit profile → Description** (750 chars max)

Lead with: *"Hello Gorgeous Med Spa — #1 Best Med Spa. Serving Aurora, Naperville, Oswego, Plainfield..."*

Include "Aurora" in the first 100 characters. Remove "day spa" if present; use "medical spa."

### 4. Change to "Appointment Required"

**GBP → Service options** → Set **Appointment required** (not "Appointment not required")

### 5. Google Posts — Aurora-Focused (1–2/week)

Post copy that says: *"10 minutes from Aurora • Botox $10/unit • Free consultation"*

Use the campaigns in `/admin/marketing/google-posts` — add "Serving Aurora" to descriptions.

### 6. Reviews (Ongoing)

**Target:** 3–5 new Google reviews per week

- Link: hellogorgeousmedspa.com/review
- Ask Aurora clients specifically: "If you drove from Aurora, we'd love a Google review!"
- Respond to every review within 24–48 hours
