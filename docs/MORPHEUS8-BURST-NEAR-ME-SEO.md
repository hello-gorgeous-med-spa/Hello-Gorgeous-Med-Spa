### 1. Google "Can't Determine Location"

When you see **"Your search did not match any documents"** or **"Can't determine location"**, Google doesn't know where you are. "Near me" searches require location.

**Fix (user-side):**
- Enable location for Google (Chrome: Settings → Privacy → Location)
- Turn off VPN if you're using one
- Don't use incognito for local searches (or enable location in incognito)
- Make sure you're signed into Google with a profile that has location
