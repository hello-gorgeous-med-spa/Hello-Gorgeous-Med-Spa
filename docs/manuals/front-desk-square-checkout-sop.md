# Front Desk — Square appointment → Terminal checkout

**Hello Gorgeous Med Spa · Oswego**  
Laminate this page. Train every desk opener. Fresha habits do not apply.

## Golden rule

**Every paid visit is checked out from the appointment — never from a blank POS sale.**

Square does **not** auto-push calendar appointments into Terminal. Checkout must start on the booking so tipping, reporting, and the appointment stay tied together.

```
Calendar appointment → Review and Check Out → Cart (prefilled) → Charge → Terminal
```

## Staff path (iPad / phone at front desk)

1. Open **Square Appointments** **or** Square POS with **Bookings / appointments** enabled.  
   Use the **same location** as the Terminal (Hello Gorgeous / `L3QDRS4DX9ZE4`).
2. Tap **today’s appointment** on the calendar (not “+ New sale”).
3. Tap **Review and Check Out**.
4. Confirm the cart shows the booked service(s). Add retail, tip, membership, or package add-ons if needed.
5. Tap **Charge** → Terminal should light up for the guest.
6. After payment succeeds, mark the appointment **Completed**.

### Backup (browser Dashboard)

**Appointments → Calendar →** open appointment → **Take Payment** → Charge → Terminal.

## Package / prepaid follow-ups

| Situation | What to do |
|---|---|
| Guest buys a package (e.g. Morpheus8 Burst ×3) | Check out the **package SKU** from that appointment once. |
| Guest returns for a prepaid session | Book **Prepaid Package Visit — $0** (or unpaid follow-up) on the customer card; note “Pkg visit 2 of 3” in seller note. Do **not** re-charge the full package. |
| Cart empty / wrong service | Fix the appointment service first, then Review and Check Out again. |

## Quick checks if cart is empty or Terminal is silent

- [ ] Terminal paired to the **correct location** (same as Appointments).
- [ ] Service has a **price** and is sellable (not a $0 ghost unless intentional prepaid visit).
- [ ] You started from the **appointment**, not a blank POS cart.
- [ ] Device signed into the **same Square account** as the booking site.
- [ ] Wi‑Fi / Terminal online; try Dashboard **Take Payment** as backup.

## Daily desk rhythm

| When | Action |
|---|---|
| Before first guest | Confirm Terminal paired + test Charge on a $0.01 or open appointment cart then cancel. |
| Every checkout | Appointment → Review and Check Out → Terminal. |
| End of day | In Hub or Square calendar, scan for **accepted/completed visits with no payment** and close them out. |

## 10-minute huddle checklist (Marissa / Michelle / desk lead)

1. [ ] Walk one live appointment: open → Review and Check Out → show Terminal prompt (cancel if demo).
2. [ ] Show the **wrong** path once: blank New Sale — explain why calendar and Terminal then “don’t sync.”
3. [ ] Point to package SKUs (Morpheus8 Burst — 3 Session Package at top of Body Contouring).
4. [ ] Confirm calendar **Color code by Service** is on (vitamins pink, weight loss red, FlowWave purple, devices orange, laser sky blue, lashes lavender).
5. [ ] Assign who audits unpaid completed appointments at close.
