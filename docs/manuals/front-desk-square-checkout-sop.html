Square checkout: Calendar → Terminal

Hello Gorgeous Med Spa · Oswego · Same location as Terminal (L3QDRS4DX9ZE4)

Golden rule: Check out from the appointment. Never open a blank “New sale” for a booked visit.
Appointment → Review and Check Out → Cart → Charge → Terminal → Mark Completed

Staff path (every guest)

  1. Open Square Appointments (or POS with Bookings mode) — same location as Terminal.
  2. Tap today’s appointment (not + New sale).
  3. Tap Review and Check Out.
  4. Cart should prefill booked service(s). Add tip / retail if needed.
  5. Charge → Terminal lights up → then mark appointment Completed.

Backup: Dashboard → Calendar → appointment → Take Payment.

If cart is empty or Terminal silent: wrong location pairing · $0/ghost SKU · blank POS cart instead of appointment checkout · wrong Square account · Terminal offline.
